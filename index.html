Zigma Builder — USD ↔ Revit 2027 · Pre-Beta

Current version 5.6.12.5 (released on the Zigma product website 2026-06-15). Daily builds keep fixing bugs and adding features.

Key Features

Revit Model Round-Trip (Model A → USD → Model B)

Powered by the Zigma V5C Engine (fidelity conversion engine), with broad support for the Revit BIM and IFC 4 specifications. A source Revit model (Model A) is exported to USD and re-imported as a Revit model (Model B) that matches the original at 99% similarity — verified across all six Revit 2027 Snowdon sample models (Architecture, Structural, HVAC, Plumbing, Electrical, Site).

IFC ↔ USD Conversion

Bidirectional IFC → USD and USD → IFC conversion at 99%+ accuracy, built directly into the same Zigma V5C fidelity engine.

Arbitrary USD Import into Revit

The Zigma Geometric Engine (model recognition engine) imports general-purpose USD files into native Revit models, with broad support for the NVIDIA AI Factory (AIF) specification. NVIDIA DSX and NVIDIA Warehouse/Factory scenes convert with 90%+ automation.

NVIDIA DSX Coverage

An NVIDIA DSX scene resolves into its 15 USD file types — Architecture, MEP, Structural and more — each recognized and rebuilt as native Revit elements.

Support Features

Built-in USD Viewer

The Zigma Builder Revit plugin ships with a built-in USD viewer (Hydra + Storm engine), so you can quickly inspect an exported USD model without leaving Revit.

Remote Bug Submission

Submit a bug ticket straight to the Zigma backend — logged and tracked, with attachments up to 50 MB. No email, phone call or instant messenger required.

Remote Model Scan Submission

One-click scan of a Revit model to surface any elements or features that cannot be converted or would be lost, submitted directly to the Zigma backend.

Real-Time Remote Debug Session

Open a 30–60 minute live remote session that lets the Zigma team chat with you directly and access your Revit model to scan, verify and run system checks for issues.

Protection & Licensing

Every Zigma EXE/DLL binary is protected in the Pre-Beta build. Licensing is fully copy-and-paste — machine code in, license key out — with no internet connection required, a fit for air-gapped fab IT policies that allow no external network.
